# 학교 보조배터리 대여 웹 (학생용 + 관리자용)

학생이 학생용 화면에 접속한 뒤 학번/이름과 배터리 번호(숫자)를 입력해 대여/반납하고, 관리자가 전체 현황을 모바일에서 확인할 수 있는 간단한 웹 예제입니다.

## 구성

- `student.html`: 학생용 화면 (학번/이름 + 배터리 번호 입력, 대여/반납)
- `admin.html`: 관리자 화면 (통계, 대여 현황, 로그 조회)
- `styles.css`: 공용 모바일 UI 스타일
- `app.js`: 공용 로직 (localStorage 기반 데이터 저장)

## 실행 방법

정적 파일 서버로 실행하면 됩니다.

### 방법 1: VS Code Live Server

- `student.html` 또는 `admin.html` 우클릭 후 Live Server로 실행

### 방법 2: Python 서버

```bash
python -m http.server 5500
```

브라우저에서 아래 주소 접속:

- 학생용: `http://localhost:5500/student.html`
- 관리자용: `http://localhost:5500/admin.html`

## 관리자 전환 안내

- 학생 화면에서 관리자 화면으로 전환할 때 비밀번호 `0000` 입력이 필요합니다.
- 관리자 페이지를 직접 열어도 비밀번호 검증 후에만 접근됩니다.

## 기본 동작 규칙

- 기본 배터리 20개(`1`~`20`)가 생성됩니다.
- 학생이 대여하면 6시간 뒤 반납 예정 시간으로 기록됩니다.
- 학생 반납은 본인 학번/이름으로 대여한 건만 가능합니다.
- 관리자는 조회만 가능하며, 강제 반납/개별 제어 기능은 없습니다.
